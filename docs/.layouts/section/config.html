Configuring Sourcegraph


Sourcegraph may be run with command line flags (run src serve -a to list options) and/or configuration which can be set in a config.ini file.

Our cloud install script and linux packages will create a default config file at /etc/sourcegraph/config.ini.

Here is a sample configuration in TOML format:

[serve]
; Sets the HTTP port of your Sourcegraph server.
http-addr = :3080
; Sets the URL of your Sourcegraph server (point DNS records to the host IP address).
app-url = http://src.mycompany.com:3080

The src CLI provides additional ways to interact with and customize Sourcegraph to fit your needs.

These articles provide detailed instructions for configuring your server.
