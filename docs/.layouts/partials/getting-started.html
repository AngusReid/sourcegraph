Getting started with Sourcegraph


Sourcegraph is a self-hosted Git repository service with Code Intelligence.

Install on Mac OS X

curl -O -L https://sourcegraph.com/.download/latest/darwin-amd64/src.gz
gunzip src.gz
chmod +x src
./src serve

Install on Linux

curl -O -L https://sourcegraph.com/.download/latest/linux-amd64/src.gz
gunzip src.gz
chmod +x src
./src serve
