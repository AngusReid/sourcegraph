gosrc.org: Go documentation site

gosrc.org is like godoc.org, with:

  • fully-linked source code for each definition (with tooltips);
  • cross-references (usage examples) for each definition; and
  • multiple revision support (all branches, tags, and commits).

Currently not all custom import paths are supported. File an issue to report problems.

How to use: Visit gosrc.org/my/pkg/import/path or use the {{template "gosrcBookmarklet"}} bookmarklet on GitHub or godoc.org.

Try it out:

  • gosrc.org/github.com/coreos/rocket/pkg/tar
  • gosrc.org/reflect
  • gosrc.org/golang.org/x/net
  • gosrc.org/sourcegraph.com/sourcegraph/appdash
