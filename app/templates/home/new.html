A new for your



Use src to create a new repository.

				$ src --endpoint={{.AppURL}} login 
				$ src repo create my/repo

Then git push from your repo directory.

				$ cd ~/path/to/my/repo
				$ git remote add origin {{.AppURL}}/my/repo
				$ git push -u origin master

Or, start from scratch!

				$ git clone {{.AppURL}}/my/repo
