Authentication

Authenticating via the command-line is easy, and will let you perform actions like creating new repositories via the src command.

Install the src program on your development machine if you haven't already.

src --endpoint={{.AppURL}} login

Endpoint

To specify which Sourcegraph server a CLI command will use, set the src --endpoint=URL option.

To create a new repository on this endpoint you would run:

$ src --endpoint={{.AppURL}} repo create my/repo

Then my/repo can be viewed in the browser at {{.AppURL}}/my/repo.
