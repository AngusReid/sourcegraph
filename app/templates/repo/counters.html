Repository view counters

Add these to the README file or your website to see how popular your repository is. Each time someone visits your GitHub repository or your website, the counter will increment by 1.

If you add both counters to the same page, visitors will be double-counted. To avoid this, append a ?no-record=1 querystring to one of the image URLs.
